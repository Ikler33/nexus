# Предложения связей (`src-tauri/src/suggest` + ai-panel «Связи»)

> Срез Ф1-9 (§6). **Режим 1 (max-sim)** — автоматический, тихий (REVIEW С-8: на save LLM не дёргаем).
> Режим 2 (LLM-обоснование) и кэш-таблица `link_suggestions` — позже (см. `docs/BACKLOG.md`).

## Алгоритм (max-sim)
`get_link_suggestions(reader, vectors, path, limit) -> Vec<LinkSuggestion>`. Считается **на лету из уже
сохранённых в usearch векторов** (embedder-сервер НЕ нужен):
1. `file_id` + его `chunk_id` + множество уже связанных файлов (исходящие+входящие `links`) — их исключаем.
2. На каждый чанк файла: `VectorIndex::get_vector` → `search_filtered(NEIGHBORS_PER_CHUNK, !own_chunk)`
   (соседи мимо чанков самого файла).
3. Агрегация по целевому файлу по **МАКСИМУМУ** similarity (max-sim): лучший чанк задаёт score и «причину».
4. Отбрасываем сам файл и связанные; порог `MIN_SCORE` (0.55, калибровка → eval Ф1-10); сорт по score↓,
   тай-брейк по пути; усечение до `limit`.

`LinkSuggestion { path, title, score, reason }` (camelCase), `reason` — сниппет лучшего совпавшего чанка.
Команда `get_link_suggestions(path, limit?)` (потолок 20); без векторов (RAG off) → пусто.

## Фронт (вкладка «Связи» в AI-панели)
- `AiPanel` — общая правая панель с вкладками **Чат** (Ф1-8) и **Связи** (Ф1-9); владеет табами, бейджем
  «локально», действием (очистить/пересчитать), закрытием. Тело — `ChatView` или `SuggestView`.
- `stores/suggest.ts`: `items` для активного файла, `load` (она же «пересчитать»), `dismiss` (прячет,
  сессия — не вернётся при пересчёте), `accept` (дописывает `[[wikilink]]` в активный буфер + dismiss).
- `SuggestView`: карточки (path, score %, «причина») + **Добавить/Скрыть**; пусто/loading/нет-файла.
  Загрузка лениво при показе вкладки и смене активного файла.
- Команда `view.suggest` (открыть панель на вкладке «Связи»); контракт `tauriApi.suggest.forFile` + мок.

## Связанный фикс (Ф0-5 редактор)
`accept` меняет буфер открытого файла → редактор должен отразить это. `Editor` синкал документ только
при СМЕНЕ файла; добавлен синк **внешнего изменения того же файла** (accept / watcher-reload) с
`externalSync`-аннотацией (без ложного dirty; эхо собственной правки гасится сравнением с текущим доком).

## Тесты
- Rust: предлагает похожую незалинкованную (max-sim), исключает связанные, пусто для непроиндексированного.
  **Живой** (nomic :8081): для заметки про кошку топ — другая про кошек (ранжируется выше физики).
  Документирует кучкование nomic на RU (риск ADR-005).
- Фронт: стор (load/dismiss-не-возвращает/accept-вставляет-`[[link]]`), `SuggestView` (карточки + скрытие),
  `Editor` (регресс: внешний синк того же файла без dirty). Проверено в превью: карточка + Add убирает её.

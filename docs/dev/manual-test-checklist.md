## 1. Чат (RAG + общий) — Ф1

- [ ] ⌘J → вопрос «по заметкам» → ответ **со ссылками-источниками**.
- [ ] Переключатель → «Общий» → ответ **без источников** (без RAG).
- [ ] Долгий ответ не лагает (rAF-троттл), «Стоп» прерывает.

## 2. Inline-LLM в редакторе — IL-1/2/3

- [ ] Курсор после текста → **⌘I** → индикатор «✦ генерирую…» → продолжение стримится серым → **Tab** принять / **Esc** отклонить.
- [ ] **Выделить** фрагмент → всплывает тулбар → **Переписать** / **Сократить** / **Продолжить** → ghost → Tab/Esc.
- [ ] **⌘K** → «ИИ: продолжить / переписать выделение / сократить выделение».
- [ ] Без выделения «Переписать» → у курсора красная inline-ошибка (не молча).
- [ ] Chat не настроен → понятная ошибка у курсора, редактор рабочий.

## 3. Дайджест изменений — ADR-007 slice 4

- [ ] Титлбар (📰) → **Сгенерировать** → дайджест недавних правок (3–6 пунктов).
- [ ] Двойной клик «Сгенерировать» → второй раз no-op (дедуп).
- [ ] Пустой/без правок vault → «нечего суммировать» (нет мусорного дайджеста).

## 4. Поиск противоречий — CT-1/2

- [ ] Титлбар (⚖) → **Найти** → пары «A ↔ B» + тип (фактическое/мягкое/устарело) + объяснение.
- [ ] Клик по заметке пары → открывается.
- [ ] Нет конфликтов → «не найдено» (а не мусор). *(нужны заметки с реальным противоречием)*

## 5. Планировщик / StatusBar — ADR-007 slice 5/6/7

- [ ] StatusBar (внизу справа): **⚙ выполняется · ⏳ в очереди · ⚠ ошибка**.
- [ ] **Backpressure (S5):** запусти дайджест → сразу печатай в чате / жми ⌘I → дайджест ждёт (**⏳**), интерактив не тормозит; стартует, когда отпустишь модель.
- [ ] **Recurring (slice 6):** после прогона дайджест/противоречия сами переназначаются (видно как новая ⏳ через сутки; быстрый sanity — не обязателен).
- [ ] **On-change (slice 7):** отредактируй пару заметок → через ~2 мин «тишины» дайджест/противоречия сами перезапустятся (StatusBar ⏳/⚙), без ручного клика.

## 6. Похожие / Связи / Цели (на новых эмбеддингах) — Ф1-9 / #35

- [ ] AI-панель → вкладка **«Похожие»** для открытой заметки (семантически близкие).
- [ ] Вкладка **«Связи»** → предложения незалинкованных + «Добавить» дописывает `[[…]]`.
- [ ] Титлбар (🎯) → **«Цели»** (заметки с инлайн `#goal` + поле `progress`).

## 7. Markdown-preview / редактор — #20 / Ф0-5

- [ ] Просмотр: заголовки/списки/таблицы/код; `[[wikilink]]` кликается, `#tag` чипом.

### Известные ограничения (ожидаемо, не баги)
- KaTeX/Mermaid в preview не рендерятся (строгий CSP) — отложено (BACKLOG).
- Поиск противоречий каждый прогон пере-судит пары (кэш — CT-3, BACKLOG).
- `continue` в inline нужен текст до курсора (на пустой странице — ошибка; используй rewrite/summarize по выделению).

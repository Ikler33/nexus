# Чек-лист ручного тестирования (на живых моделях)

> Живой документ: полный проход по приложению руками — то, что **не** покрыто офлайн-тестами
> (`cargo test`/`vitest`): качество LLM-вывода, end-to-end UX, интеграция с Tauri-webview, ощущения.
> **Правило:** каждый срез с пользовательской фичей дописывает/обновляет сюда пункт(ы).
>
> Хоткеи (macOS — ⌘, иначе Ctrl): палитра ⌘K · новая заметка ⌘N · сохранить ⌘S · граф ⌘G · чат ⌘J ·
> inline ⌘I · режим чтения ⌘R · настройки ⌘, (запасной — палитра).

## A. Каркас / Vault / дерево файлов — Ф0

- [ ] **Дерево:** ленивая подгрузка папок (раскрытие), клик по файлу открывает.

## H. Режим чтения — ⌘R

- [ ] **⌘R** прячет сайдбар/AI-панель, центрирует документ; **Esc** выходит.

## I. AI: Чат (RAG + общий) — Ф1-7/8

- [ ] ⌘J → вопрос **«по заметкам»** → ответ **со ссылками-источниками**.
- [ ] Переключатель → **«Общий»** → ответ **без источников** (без RAG).
- [ ] Долгий ответ не лагает (rAF-троттл); **«Стоп»** прерывает; история сохраняется между перезапусками (на vault).

## J. AI: Inline-LLM в редакторе — IL-1/2/3

- [ ] Курсор после текста → **⌘I** → «✦ генерирю…» → продолжение серым → **Tab** принять / **Esc** отклонить.
- [ ] **Выделить** фрагмент → тулбар → **Переписать / Сократить / Продолжить** → ghost → Tab/Esc.
- [ ] **⌘K** → «ИИ: продолжить / переписать выделение / сократить выделение».
- [ ] Без выделения «Переписать» → красная inline-ошибка у курсора (не молча). Chat не настроен → ошибка, редактор рабочий.

## K. AI: Дайджест изменений — ADR-007 slice 4

- [ ] Титлбар (📰) → **Сгенерировать** → дайджест недавних правок (3–6 пунктов).
- [ ] Двойной клик «Сгенерировать» → второй раз no-op (дедуп). Пустой vault → «нечего суммировать».

## L. AI: Поиск противоречий — CT-1/2

- [ ] Титлбар (⚖) → **Найти** → пары «A ↔ B» + тип (фактическое/мягкое/устарело) + объяснение; клик открывает заметку.
- [ ] Нет конфликтов → «не найдено» (нужны заметки с реальным противоречием).

## M. AI: Похожие / Связи / Цели — Ф1-9 / #35

- [ ] AI-панель → **«Похожие»** для открытой заметки (семантически близкие).
- [ ] **«Связи»** → предложения незалинкованных + «Добавить» дописывает `[[…]]`.
- [ ] Титлбар (🎯) → **«Цели»** (заметки с инлайн `#goal` + поле `progress`, прогресс-бары).

## N. Планировщик / StatusBar — ADR-007 slice 5/6/7

- [ ] StatusBar (внизу справа): **⚙ выполняется · ⏳ в очереди · ⚠ ошибка**.
- [ ] **Backpressure (S5):** запусти дайджест → сразу печатай в чате / жми ⌘I → дайджест ждёт (**⏳**), интерактив не тормозит; стартует, когда отпустишь модель.
- [ ] **Recurring (slice 6):** дайджест/противоречия сами переназначаются (новая ⏳ ~через сутки; sanity-чек).
- [ ] **On-change (slice 7):** отредактируй пару заметок → через ~2 мин «тишины» дайджест/противоречия сами перезапустятся (StatusBar ⏳/⚙).

- [ ] Картинка/PDF во вкладке — открывается во вьюере (не как текст).

## P. Плагины — Ф2

## Q. Git-синхронизация — Ф3

- [ ] Титлбар (ветка) → панель «Синхронизация»: список изменений → **Закоммитить** (secret-scan: при находке секрета коммит блокируется).

### Известные ограничения (ожидаемо, не баги)
- KaTeX/Mermaid в preview не рендерятся (строгий CSP) — отложено (BACKLOG).
- Поиск противоречий каждый прогон пере-судит пары (кэш — CT-3, BACKLOG).
- `continue` в inline нужен текст до курсора (на пустой странице — ошибка; используй rewrite/summarize по выделению).
- Граф на дефолтах размазан (тюнится слайдером; v2e — BACKLOG).
- Дубли воркеров при переоткрытии vault — лог-шум, не корраптит (BACKLOG: shutdown-сигнал).

# Поиск (`src-tauri/src/search` + сайдбар)

> Срез Ф0-7. Допущение Ф0: поиск по **title/path/tags** (метаданные). Полнотекстовый по телу
> (FTS5 поверх `chunks`, §5) — Фаза 1, т.к. chunks появляются там (§11.7 промпта).

## Rust
- `search_notes(reader, query) -> Vec<NoteRef>`: `files` LEFT JOIN `file_tags`/`tags`,
  `path/title/tag LIKE %q%` (спецсимволы LIKE экранируются, `ESCAPE '\'`), `LIMIT 100`.
- Команда `search_vault(query)`.

## Фронт
- `Sidebar`: поле поиска (debounce 150 мс) + дерево/результаты. Пустой запрос → `FileTree`,
  непустой → список результатов; клик → `openFile`. Вне Tauri — мок.

## Ограничения / дальше
- LIKE без ICU: кириллица сейчас регистрозависима (для ASCII — нет). Полнотекст + ранжирование
  (BM25) и кросс-язычность — Ф1 (FTS5 поверх chunks, §5/§6).

## Тесты
- Rust: поиск по path/title/tag, пустой/пробельный запрос.
- Фронт: дерево↔результаты, поиск по тегу, пустое состояние, очистка.
